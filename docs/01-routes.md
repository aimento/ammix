# 화면 구성

```
/home               : 홈 화면
/about              : 앱 정보 화면
/sign-up            : 회원 가입 화면
/sign-in            : 로그인 화면
```

- 모든 페이지는 로그인 후, 접근 가능

1. 홈 화면

- 아이디 표시

2. 앱 정보 화면

- 저작권 정보 표시

3. 회원 가입 화면

- 회원가입 입력변수: username, password, email-address
- 회원가입 완료 후, 자동 로그인 처리 및 홈 화면으로 이동

4. 로그인 화면

- username, password 입력
